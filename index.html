HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Tags are specifically what are contained in brackets
  • Elements are everything from the opening tag through to the closing tag
  • HTML stands for Hypertext Markup Language
  • HTML is one of the fundemental languages needed in the creation of web applications

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • It is important to be working on a feature branch
  • Before we push this up to our repo, we should double-check if our local branch is in sync with the base branch in GitHub. So let’s pull down any changes from the base branch... git pull origin main
  • When creating a Pull Request to merge Feature Branch we would usually request someone/team checks our code first

Terminal Git Commands to create new branch/edit/upload to GitHub

  • Check which branch you are on - git status
  • Switch to Main Branch if required - git checkout main
  • Confirm Main up to date - git pull origin main
  • Create new branch - git checkout -b feature/.......
  • Make changes to files
  • Make sure still on relevant branch - git status change if required
  • To add files to staging area of Git - git add -A
  • Now files are staged....commit them to repo - git commit -m "enter relevant notes here"
  • Make sure our current branch is up to date before opening a pull request - git pull origin main
  • Navigate to relevant branch and make sure its connecting to Main
  • Open pull request - Edit or add title - click create pull request
  • Merge the branches - Merge pull request" button and then confirming the merge request
  • Make sure we update our local main branch now. Back in the command line, navigate to the main branch by entering the following Git command: git checkout main
